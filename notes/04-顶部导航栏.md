## Header (SiteHeader)

最后，我们把 Logo、导航链接和主题切换按钮组合在一起。

在 `components` 文件夹下新建 `site-header.tsx`：

```tsx
import Link from "next/link"
import { ModeToggle } from "@/components/mode-toggle"
import { SquircleDashed } from 'lucide-react';

export function SiteHeader() {
    return (
        <header className="sticky top-0 z-50 w-full border-b border-border/40 bg-background/95 backdrop-blur supports-backdrop-filter:bg-background/60">
            <div className="container flex h-14 max-w-screen-2xl items-center mx-auto px-4">
                {/* 左侧 Logo */}
                <Link href="/" className="mr-6 flex items-center space-x-2 font-bold">
                    <SquircleDashed mr-4 />
                    <span className="hidden sm:inline-block">
                        XGWebSite-Awesome
                    </span>
                </Link>

                {/* 右侧工具栏 */}
                <div className="flex flex-1 items-center justify-between space-x-2 md:justify-end">
                    <nav className="flex items-center gap-1">
                        <ModeToggle />
                    </nav>
                </div>
            </div>
        </header>
    )
}
```
> **CSS 魔法**: `backdrop-blur` 和 `bg-background/95` 共同创造了那个高级的“磨砂玻璃”效果。当页面滚动时，内容会在导航栏下方模糊地透出来。

### TailWind 样式
- `sticky` 使导航栏固定在顶部。
- `z-50` 确保导航栏在其他内容之上。
- `w-full` 宽度占满父容器。
- `border-b` 下边框。
- `border-border/40` 边框颜色为 `border` 颜色的 40% 透明度。
- `bg-background/95` 背景颜色为 `background` 颜色的 95% 透明度。
- `backdrop-blur` 模糊背景。
- `supports-backdrop-filter:bg-background/60` 支持 backdrop-filter 时，背景颜色为 `background` 颜色的 60% 透明度。
---

- `container` 容器类，用于居中对齐和设置最大宽度。
- `flex` 弹性布局。
- `h-14` 高度为 14 个单位。
- `max-w-screen-2xl` 最大宽度为 2xl 屏幕宽度。
- `items-center` 垂直居中对齐。
- `mx-auto` 水平居中对齐。
- `px-4` 水平内边距为 4 个单位。

---
### 左侧 Logo
- `hidden` 隐藏元素（在小屏幕上）。
- `md:flex` 在中等屏幕及以上显示为弹性布局。
- `mr-6` 右侧外边距为 6 个单位。
- `flex items-center space-x-2` 弹性布局，垂直居中对齐，项目之间间距为 2 个单位。

## 5. 最后一步：把它放到 Layout 里

打开 `app/layout.tsx`，把 `SiteHeader` 加进去，放在 `children` 的上面。

```tsx
// ... 其他 import
import { SiteHeader } from "@/components/site-header";

// ... RootLayout 组件内
<ThemeProvider ...>
  <div className="relative flex min-h-screen flex-col bg-background">
    <SiteHeader />
    <main className="flex-1">
      {children}
    </main>
  </div>
</ThemeProvider>
```
